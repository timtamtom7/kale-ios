# KaleMac — Launch Checklist

### Code & Build
- [ ] All Swift files compile with zero errors

- [ ] Dark mode renders correctly (all views use theme tokens)
- [ ] macOS 14.0 compatibility verified
- [ ] Archive builds successfully for distribution

- [ ] App icon: 1024 × 1024 PNG ready
- [ ] App Store screenshots: 5 screenshots at 1512 × 982 @ 2x

### App Store Connect
- [ ] Bundle ID registered: `com.kale.kale-mac`

- [ ] Category: Food & Drink > Meal Planning
- [ ] Minimum OS: macOS 14.0
- [ ] Pricing: Free / Paid (choose one)
- [ ] Privacy policy URL posted
- [ ] Support URL posted

### Account & Signing
- [ ] Apple Developer account active
- [ ] Signing certificate created (Development + Distribution)
